# Codealpha_Chatbot
A simple rule-based chatbot built using Python. This chatbot responds to basic user messages like hello, how are you, and bye using if-elif conditions, loops, functions, and input/output concepts.

This project is a simple rule-based chatbot created using Python as part of my CodeAlpha Python Internship.

## Project Goal
Build a chatbot that responds to basic user messages with predefined replies.

## Features
- Responds to messages like:
  - hello
  - how are you
  - what is your name
  - bye
- Uses simple rule-based logic
- Beginner-friendly Python project

## Concepts Used
- if-elif conditions
- Functions
- While loop
- Input and Output

## Technologies Used
- Python

## How to Run

1. Save the file as `chatbot.py`
2. Open terminal or command prompt
3. Run the program:

```bash
python chatbot.py
